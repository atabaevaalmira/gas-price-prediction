# Gas Price Prediction Project 📈

This project was developed for the **ML Club**. It uses machine learning to predict natural gas prices.

## Overview
- **Data Source:** Yahoo Finance (via `yfinance`)
- **Model:** Linear Regression
- **Frameworks:** Pandas, Scikit-learn, Matplotlib

## Results
The model analyzes the price of the previous day to predict the next day's closing price. 
Current prediction for tomorrow: **$2.81** (based on today's $2.80).

## How to run
1. Install dependencies: `pip install pandas scikit-learn yfinance matplotlib`
2. Run the `prediction.ipynb` notebook.
